# Project

海釣り初心者向けの釣り予測ブラウザアプリ。
初期対象は以下の3地点。
- 大黒海づり施設
- 本牧海づり施設
- 東扇島西公園

# Priorities

1. まずはブラウザ版MVP
2. 初心者向けの分かりやすさを優先
3. 予測結果には必ず理由を表示
4. 機械学習ではなくルールベースで開始
5. 後からデータ取得処理を差し込みやすい構成にする

# Coding preferences

- 型安全性を重視
- UI、データ取得、スコアリングを分離
- まずはダミーデータで画面を成立させる
- 画面はスマホでも見やすいレスポンシブ構成
- 専門用語だけで終わらせず、初心者向け説明を含める

# Expected deliverables

- トップ画面
- 釣り場詳細画面
- 用語説明画面
- ダミーデータ
- 今後の実装タスク一覧
